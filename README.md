# 😀 人脸表情识别系统

基于深度学习的表情识别系统，可以识别7种人脸表情。

## 🎯 项目效果

上传人脸照片，AI自动识别表情：
- 😠 生气 | 😖 厌恶 | 😨 恐惧 | 😊 开心 | 😢 难过 | 😲 惊讶 | 😐 中性

## 📊 模型性能

| 指标 | 结果 |
|-----|------|
| 测试准确率 | **60.32%** |
| 训练集大小 | 28,709张 |
| 测试集大小 | 3,589张 |
| 模型参数量 | 约50万 |

## 📊 模型性能

| 模型 | 测试准确率 | 参数量 | 说明 |
|-----|-----------|-------|------|
| 基础CNN | 60.32% | ~50万 | 3层卷积+Dropout |
| **改进版CNN** | **65.65%** | ~150万 | 4层卷积+GAP+数据增强 ✅ |

**改进提升：+5.33%**
