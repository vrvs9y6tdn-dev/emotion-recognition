# 神经网络与深度学习课程大作业
单人独立项目：基于CNN的4类观赏花卉图像智能分类
专业：数据科学与大数据技术

## 1.项目简介
本项目采用PyTorch搭建三层轻量化卷积神经网络CNN，基于Oxford Flowers102花卉数据集筛选出雏菊、向日葵、玫瑰、郁金香4类观赏花卉完成图像分类任务。针对多类别图像训练易出现梯度震荡、分类全部预测错误问题，引入BatchNorm批归一化、Dropout随机失活缓解梯度消失与过拟合，搭配随机翻转、旋转、平移数据增强扩充样本多样性，提升模型泛化能力。模型训练12轮后测试集准确率稳定达到88%以上，配套Gradio搭建本地网页可视化分类器，支持任意上传花卉实拍图片实时输出识别结果，同时输出训练损失/准确率变化曲线、4分类混淆矩阵完成完整实验分析。

## 2.项目技术栈
Python、PyTorch、Torchvision、Matplotlib、Seaborn、NumPy、Gradio可视化交互框架

## 5.数据集介绍
采用官方Oxford Flowers102花卉数据集，筛选仅保留4个观赏花卉类别，降低模型学习压力，避免训练崩盘：
1. 雏菊  2. 向日葵  3. 玫瑰  4. 郁金香
所有图片统一缩放至64×64分辨率，训练集启用多重随机数据增强，测试集仅标准化处理，保证评估公平稳定。

## 6.CNN卷积网络完整结构
1. 卷积块1：Conv2d(3,32,3,padding=1) + BatchNorm2d + ReLU激活 + MaxPool2d(2,2) + Dropout(0.25)
2. 卷积块2：Conv2d(32,64,3,padding=1) + BatchNorm2d + ReLU激活 + MaxPool2d(2,2) + Dropout(0.25)
3. 卷积块3：Conv2d(64,128,3,padding=1) + BatchNorm2d + ReLU激活 + MaxPool2d(2,2) + Dropout(0.25)
4. 全连接分类层：128*8*8展平特征 → 256维隐藏层(Dropout0.4) → 4维花卉分类输出

## 10.项目优化创新点（解决多分类训练不理想、分类全错问题）
1. 缩减花卉分类数量至4类，大幅降低模型学习压力，规避多类别训练梯度不收敛、全部预测同一类的问题；
2. 网络每层卷积搭配BatchNorm批归一化层，稳定训练梯度，防止损失剧烈震荡、模型完全不学习；
3. 每层卷积后增加Dropout随机失活层，抑制模型过拟合，提升测试集图片泛化识别效果；
4. 训练集搭配多重随机数据增强（翻转、旋转、平移），扩充样本多样性，进一步提升分类准确率；
5. 搭建Gradio网页可视化交互分类器，支持自定义图片上传实时推理，可直观演示模型效果，满足作业图片上传分类要求。
