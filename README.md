# 六类陆生动物图像分类项目
## 项目简介
基于三层CNN卷积神经网络，使用CIFAR10官方数据集筛选6类陆生动物（鸟类、猫、鹿、狗、马、青蛙）完成彩色图像分类。
## 环境依赖
pip install -r requirements.txt
## 运行步骤
1. 打开src/animal_classify.ipynb
2. 从上至下依次运行单元格
3. 训练完成自动弹出Gradio网页，可上传动物图片推理分类
## 项目功能
1. 数据增强：随机旋转、翻转、裁剪
2. 训练可视化：损失曲线、准确率曲线
3. 拓展模块：混淆矩阵、错分样本分析、模型权重保存
## 数据集
CIFAR10 公开标准彩色图像数据集，自动国内镜像下载
